# Multimodal Interview Assessment System

A deep learning web app that analyzes interview performance across three dimensions:
- Facial emotion detection (DeepFace + OpenCV)
- Speech-to-text + voice analysis (Whisper + librosa)
- Text sentiment + NLP analysis (HuggingFace + spaCy)

## Live Demo
https://interviewtest-shristiyoruichi.streamlit.app/

## Tech Stack
Python, TensorFlow, PyTorch, Streamlit, OpenAI Whisper, HuggingFace Transformers, spaCy, librosa, DeepFace

## Modules
- Module 1: Facial Emotion Analysis
- Module 2: Speech to Text
- Module 3: Text & Sentiment Analysis
- Module 4: Voice Analysis
- Module 5: Weighted Scoring Engine
